# 🐍 Python Projects

Welcome to my **Python Projects** repository! This repository contains beginner-friendly Python projects that helped me practice programming fundamentals and improve my problem-solving skills.

## 📂 Projects Included

### 🎲 Coin Toss Game
- Simulates a coin flip (Heads or Tails)
- Uses the `random` module
- Demonstrates conditional statements

### 🔢 Number Guess Game
- Guess the randomly generated number
- Gives hints if the guess is too high or too low
- Uses loops and conditions

### 🔐 Password Guess Game
- User guesses the correct password
- Demonstrates loops, input handling, and conditional logic

### ✂️ Rock Paper Scissors
- Play against the computer
- Uses random choices and game logic
- Demonstrates decision-making with `if-elif-else`

### 🏦 Bank Management System
**Features:**
🏦 Create New Account
Create a new customer account.
Automatically generates a unique account number.
Supports Savings and Current account types.
Stores customer information in CSV files.

💰 Deposit Money
Deposit money into any active account.
Updates the account balance instantly.
Saves the transaction history.

💸 Withdraw Money
Withdraw money from an account.
Prevents withdrawal if there is insufficient balance.
Records every withdrawal transaction.
🔄 Transfer Money
Transfer funds between two active accounts.
Updates both sender's and receiver's balances.
Creates transaction records for both accounts.
📊 Check Account Balance
View current account details.
Displays:
Account Number
Customer Name
Account Type
Available Balance
📜 Transaction History
View complete transaction history of an account.
Shows:
Transaction ID
Date & Time
Transaction Type
Amount
Balance After Transaction
Transaction Details
👥 List Active Accounts
Displays all active customer accounts.
Shows account number, customer name, account type, balance, and creation date.
❌ Close Account
Close an account only when its balance is zero.
Changes account status from Active to Closed.
Records the account closure in transaction history.
💾 CSV File Storage
Stores account information in accounts.csv.
Stores transaction records in transactions.csv.
No database installation required.
🔢 Automatic Account Number Generation
Generates sequential account numbers starting from 1001.
🆔 Unique Transaction ID
Every transaction is assigned a unique ID using UUID.
🕒 Date & Time Logging
Every account creation and transaction is saved with the current date and time.
✅ Input Validation
Prevents invalid account numbers.
Prevents negative or zero transaction amounts.
Prevents overdrawing from an account.
Prevents transfers to the same account.
📂 Automatic Data Folder Creation
Creates a bank_data folder automatically if it does not exist.
🔄 Persistent Data
All account and transaction data is saved permanently in CSV files and remains available after restarting the program.
